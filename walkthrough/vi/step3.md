# Luôn sử dụng mô hình AI tốt nhất

**Kết nối API key của bạn cho Anthropic (Claude), Google (Gemini), OpenAI (GPT) và các LLM hàng đầu khác.**

Cline đặt bạn ở vị trí tiên phong của AI. Mang theo API key của riêng bạn cho các mô hình hàng đầu như Anthropic (Claude), Google (Gemini) và OpenAI (GPT). Luôn tận dụng khả năng State-of-the-Art (SOTA) mạnh mẽ nhất, đảm bảo bạn kiểm soát cả chi phí và hiệu suất tiên tiến.
