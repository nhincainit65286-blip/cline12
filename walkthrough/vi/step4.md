# Mở khóa khả năng chuyên biệt với MCP

**Model Context Protocol (MCP) kết nối Cline với thế giới các công cụ mạnh mẽ.**

Vượt ra ngoài code local. Với Model Context Protocol (MCP), Cline truy cập ngữ cảnh quan trọng từ các nguồn dữ liệu bên ngoài như database và API. Nó có thể tương tác với các nền tảng này và tận dụng marketplace đang phát triển của các công cụ chuyên biệt, an toàn để giải quyết các tác vụ phát triển phức tạp, thực tế.
